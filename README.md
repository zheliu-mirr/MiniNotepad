# MiniNotepad

轻量级 Windows 文本处理工具，比记事本更强，比 IDE 更轻。

## 特性

- **记事本完整功能** — 新建/打开/保存/查找替换/跳转行/字体设置/自动换行，你熟悉的操作一个不少
- **多行批量处理** — 核心功能，选中多行后自由选择添加前缀/后缀，支持所有行/奇数行/偶数行/自定义行范围
- **默认编辑器** — 一键替换 Windows 自带记事本，双击 .txt 文件直接用 MiniNotepad 打开
- **免安装** — 单个 EXE 文件，复制即用，不写注册表（设为默认编辑器除外）
- **体积小** — 不依赖 chardet 也能自动检测编码（UTF-8/GBK/Big5/BOM 等）
- **Apache 2.0 开源** — 随便用，随便改

## 批量处理功能

通过菜单 `批量处理` 或快捷键 `Ctrl+B` 打开：

| 操作 | 说明 |
|------|------|
| 添加前缀/后缀 | 给指定行首或行尾插入文本 |
| 删除前缀/后缀 | 移除行首或行尾的匹配文本 |
| 替换文本 | 对指定行进行文本替换 |
| **锚点前插入** | 在指定锚点文本前插入内容（如 `123 封禁xxx`） |
| **锚点后插入** | 在指定锚点文本后插入内容 |
| **锚点前删除N字符** | 删除锚点前指定数量的字符 |
| **锚点后删除N字符** | 删除锚点后指定数量的字符 |
| 删除空行 | 一键清理所有空行 |
| 删除重复行 | 保留首次出现的行，去除后续重复 |
| 行排序 | 升序或降序排列 |
| 行倒序 | 整体翻转行顺序 |
| 添加/去除行号 | 自动编号或移除行号 |
| Trim | 去除每行首尾空格 |
| 大小写转换 | 全部转大写或小写 |

行选择模式：

- **所有行** — 对全部行生效
- **奇数行** — 只处理第 1, 3, 5... 行
- **偶数行** — 只处理第 2, 4, 6... 行
- **选中文本所在行** — 只处理当前选中的行
- **自定义行范围** — 如 `1-5,8,10` 处理第1到5行、第8行、第10行

## 构建

需要 Python 3.8+，双击 `build.bat` 即可：

```
build.bat
```

生成的 EXE 位于 `dist\MiniNotepad.exe`。

## 设为默认文本编辑器

1. 先用 `build.bat` 打包出 `MiniNotepad.exe`
2. 以管理员身份运行 `MiniNotepad.exe`
3. 菜单 → 帮助 → 设为默认文本编辑器

之后双击任何 .txt 文件都会用 MiniNotepad 打开。

## 运行要求

- **构建时**：Windows + Python 3.8+
- **运行时**：Windows 10/11，无需安装 Python 或任何依赖
