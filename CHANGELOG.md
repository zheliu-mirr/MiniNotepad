# Changelog

## 2025-06-12

### 新增
- 完整的 Windows 记事本功能（新建/打开/保存/另存为/查找替换/跳转行/字体设置）
- 多行批量处理系统，支持 14 种操作：
  - 添加前缀/后缀、删除前缀/后缀、替换文本
  - **锚点前插入** — 在指定文本前插入内容
  - **锚点后插入** — 在指定文本后插入内容
  - **锚点前删除N字符** — 删除指定文本前 N 个字符
  - **锚点后删除N字符** — 删除指定文本后 N 个字符
- 5 种行选择模式：所有行/奇数行/偶数行/选中文本所在行/自定义行范围
- 快捷批量操作：删除空行/删除重复行/行排序/行倒序/添加去除行号/Trim/大小写转换
- 行号显示与隐藏切换
- 状态栏显示行列信息/总行数/字数/编码/缩放比例
- Ctrl+滚轮缩放字体大小
- 一键设为 Windows 默认文本编辑器（注册表 .txt 关联 + IFEO 替换 notepad.exe）
- 多编码自动检测（BOM / chardet 可选 / 常见编码回退）
- 全局异常捕获，崩溃时弹窗显示完整错误信息并写入桌面日志

### 修复
- 修复缩放功能永久修改字体大小的 bug（基于 base_font_size 偏移）
- 修复查找下一个每次都弹出新窗口的问题（记住搜索状态，F3 直接跳转）
- 修复打印功能调用不存在方法的问题（改用 notepad /p 打印）
- 修复行号区域滚动不同步的问题
- 修复 IFEO 模式下 Windows 传入 notepad.exe 路径导致误打开文件的问题
- 修复 `<MouseWheel>` 事件在 Windows 上不识别导致启动崩溃的问题（跨平台兼容处理）
- 修复批量处理对话框按钮被高 DPI 截断不可见的问题

### 构建
- PyInstaller 单文件打包，免安装运行
- `build.bat` 一键构建脚本
- Apache 2.0 开源协议
